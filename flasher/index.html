Legacy Bridge USB Flasher

Безопасная первичная прошивка ESP32-C3 по USB прямо из браузера (Chrome/Edge, HTTPS).

USB flasher ставит полный образ прошивки: bootloader, partitions, System, LittleFS и Recovery.

Для первичной прошивки выбирайте полное стирание, чтобы старые NVS/OTA-data не мешали старту.

Как прошивать:

        
1. Подключите ESP32-C3 по USB.

        
2. При необходимости войдите в boot mode: удерживайте BOOT, нажмите RESET, отпустите BOOT.

        
3. Выберите версию и нажмите кнопку установки. Согласитесь на erase, если браузер спросит.

        
4. Дайте браузеру доступ к serial-порту устройства.

        
5. После перезагрузки устройство должно стартовать в рабочей прошивке выбранной версии.
