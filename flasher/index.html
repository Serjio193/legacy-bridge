Legacy Bridge USB Flasher

Прошивка "голого" ESP32-C3 по USB прямо из браузера (Chrome/Edge, HTTPS).

Источник: GitHub Releases репозитория Serjio193/legacy-bridge.

Загрузка списка релизов...

Как прошивать:

1. Подключите ESP32-C3 по USB.

2. При необходимости войдите в boot mode: удерживайте BOOT, нажмите RESET, отпустите BOOT.

3. Выберите версию и нажмите кнопку установки.

4. Дайте браузеру доступ к serial-порту устройства.
